# Features
  - Make remote networking calls to retrieve data
  - Display video via ExoPlayer
  - Espresso testing
  - Homescreen widget
  - Dynamically created tabbed viewpager
  - Recyclerviews
  - Activity/Fragment lifecycles rotation handling
  - Multi-pane for tablets
  - Activity intents

# Libraries
